Используем псевдокод(описание) для проектирования игры:

Выбрать случайное слово
Пока(while) слово не угадано {
  Показать(alert) игроку текущее состояние игры
  Запросить(prompt) у игрока вариант ответа
  Если(if(break),else if,else) игрок хочет выйти из игры {
    Выйти из игры
}
Иначе Если(else if) вариант ответа — не одиночная буква {
  Сообщить игроку, что он должен ввести букву
}
Иначе(else) {
  Если такая буква есть в слове {
    Обновить(for) состояние игры, подставив новую букву
    }
  }
}
Поздравить(alert) игрока с победой — слово угадано и отобразить(alert) на экране.
