*** ПЕРЕМЕННЫЕ которые понадобятся!!!

1. Обявить три переменные для хранения позиции каждой клетки корабля.
Присвоить именна переменных location1, location2, location3.

2. Обявить переменную для номера ТЕКУЩЕЙ ПОПЫТКИ.Присвоить ей имя - guess.

3. Объявить переменную для К-ВА ПОПАДАНИЙ.Присвоить ей имя - hits и 
инициализировать значением 0.

4. Объявить переменную для К-ВА ПОПЫТОК.Присвоить ей имя - guesses и 
инициализировать значением 0.

5. Объявить переменную для ХРАНЕНИЯ ИНФОРМАЦИИ о том, потоплен корабль или нет.
Присвоить ей имя iaSunk и инициализировать значением false.

*** ПРОГРАМННАЯ логика!!!

Запускаем ЦИКЛ while {

         Запрашиваем координаты выстрела от пользователя (через -- prompt)
            Сравниваем введеные данные с диапазоном корректных значений (через -- if)
            {
            Если введены данные не корректно,ВЫВЕСТИ(через -- alert) сообщение 
               с предложением ввести согласно предложеного диапазона 
            }       
        { 
        Иначе,ПРИБАВИТЬ один к guesses
        } 
            {
            Если выстрел попал в цель:
                       ПРИБАВИТЬ один к количеству попаданий к hits
            }
        {
            Если количество попаданий равно ТРЕМ
            ПРИСВОИТЬ isSunk значение true
        }

        {
            ВЫВЕСТИ сообщение "Корабль потоплен!"
        }    
} 
ВЫВЕСТИ статистику за какое К-ВО ПОПЫТОК удалось потопить корабль.
